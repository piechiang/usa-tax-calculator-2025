# 🏦 美国税务计算器 2025

一个功能完整的2025年美国联邦税和马里兰州税计算器

## ✨ 功能特点

- 📊 **准确计算** - 使用2025年最新税率和标准扣除额
- 📋 **全面支持** - 支持所有报税身份类型
- 🏛️ **州税计算** - 马里兰州税和地方税计算
- 📄 **表格生成** - 生成标准1040表格PDF
- 💾 **数据导出** - 支持税务数据导出
- 🎨 **现代界面** - 响应式设计，支持移动设备
